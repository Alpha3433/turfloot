<product_requirements>
TurfLoot is envisioned as an Agario-style, real-time multiplayer game with integrated Solana SOL blockchain features. The application requires secure Privy.io authentication, a dark UI, a live pot ticker, user profiles, and dynamic game statistics. Core gameplay involves player movement, orb collection for growth, virus collision, and a cash-out system with interactive popups. The development trajectory has focused on evolving the app from bot-based to a real-time multiplayer experience, necessitating smart contract integration and server-side anti-cheat measures. Specific UI/UX enhancements include unique usernames, character customization, dynamic player balance/mass display, fixed mission tracking, a functional server browser, server indicators, dynamic Coins Collected and Time Alive stats, and a redesigned, high-contrast in-game minimap. Significant effort has been put into ensuring robust authentication, refining visual elements like the landing page background, and achieving deployment readiness, despite external infrastructure challenges affecting the live blockchain balance display. The latest requirement is to build a 2-player lobby system, visible on the landing page, with detailed specifications for UI, backend, real-time events, and game allocation.
